Presentación interactiva (HTML autónomo) servida con Streamlit.

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Estructura
- `app.py` — app Streamlit que renderiza la presentación
- `presentacion.html` — presentación completa (sin dependencias externas)
